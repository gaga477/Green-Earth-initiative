# 🌍 Green Earth Initiative

A **real-world eco gamification platform** that turns environmental actions into rewards.

Users complete real-life environmental tasks, submit proof, earn points, and climb a global leaderboard — building a greener planet through technology.

---

## 🚀 Project Overview

Green Earth Initiative is a full-stack system consisting of:

- 🎮 Web Game (React)
- 🧑‍💻 Admin Panel (React)
- 📱 Mobile App (Flutter)
- 🔙 Backend API (Node.js + Express)
- ☁️ Database (MongoDB / Firebase-ready)

It combines:
- Environmental sustainability 🌱  
- Gamification 🎮  
- Reward economy 💰  
- Social impact 🌍  

---

## 🧠 Core Idea

> “Do real-world eco tasks → Submit proof → Earn points → Get rewarded”

Example tasks:
- Pick up waste 🗑️
- Plant trees 🌳
- Community cleanups 🧹
- Recycling activities ♻️
